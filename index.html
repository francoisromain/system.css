A minimal CSS framework

System.css combine a css utility library and a few postcss plugins:

  • postcss-normalize
  • cssnext (imported as separate plugins, to be up to date)
  • postcss-grid-system
  • postcss-grid-fluid
  • postcss-typescale
  • postcss-button

Utils

A utility class for positionning: clearfix, hide, absolute, full size, etc.

Margins

A margin utility. Syntax: m[direction]-[size].

  • m (for margin)
  • Direction (optional): t (top), r (right), b (bottom), l (left), x (left and right), y (top and bottom)
  • Size (optional). Values for each size is defined in the settings: 0 (no margin), xxs, xs, s, m, (default), l, xl, xxl, auto.

Paddings

A padding utility. Syntax: p[direction]-[size].

  • p (for padding)
  • Direction (optional): t (top), r (right), b (bottom), l (left), x (left and right), y (top and bottom)
  • Size (optional). Values for each size is defined in the settings: 0 (no padding), xxs, xs, s, m, (default), l, xl, xxl.
